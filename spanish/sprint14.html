Insights basados en datos para una app de reseñas de libros: Proyecto SQL Analytics

Analicé información de libros, autores, editoriales, calificaciones y reseñas utilizando SQL para descubrir los principales insights del negocio. El análisis se centró en publicaciones posteriores al año 2000, autores mejor valorados y usuarios con mayor participación. Se entregaron métricas clave para apoyar la estrategia de producto de una app de lectura competitiva.

Conclusión

El análisis en SQL reveló insights clave para una startup de libros: se identificaron 819 libros publicados después del año 2000, con títulos destacados como “13 Little Blue Envelopes”, que obtuvo una calificación promedio de 4.67. Penguin Books lideró en cantidad de publicaciones, mientras que J.K. Rowling/Mary GrandPré tuvo los libros mejor valorados (4.28 con más de 50 reseñas). Los usuarios más activos escribieron en promedio 24.33 reseñas, mostrando un alto nivel de compromiso —factor clave para impulsar la retención y las decisiones estratégicas del producto.
